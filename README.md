## 📖 概述

数控铣床面临两大挑战：
- **时序分布漂移**：服役 10~15 年间，机械磨损与润滑退化导致振动信号统计特性逐渐改变。
- **跨工况异质性**：频繁的生产任务切换与加工参数调整带来显著的数据分布差异。

TCB-Net 采用**双聚类**架构解决上述问题：
- **时间聚类**：利用 1D-CNN 提取特征，通过带指数移动平均（EMA）的在线 K-means 跟踪分布漂移。
- **通道聚类**：基于皮尔逊相关系数与能量比，建立正常耦合、不对中、切屑夹持、卡盘/冲击四种物理模式，并进行软分配。
- **双聚类融合**：将时间隐特征与通道权重拼接，经残差块、自注意力和 MLP 输出故障概率，采用 Focal Loss 应对类别不平衡。

模型轻量（约 5 万参数）且具有良好的可解释性，可提供基于簇的风险分析。

## ⚙️ 方法

**三个核心模块：**

1. **时间聚类模块**  
   - 使用三层 1D-CNN + 全局平均池化提取时序特征。  
   - 将每个滑动窗口分配到 `K` 个时间簇之一，中心通过 EMA 在线更新。

2. **通道聚类模块**  
   - 计算三轴信号的皮尔逊相关系数（`r_xy, r_xz, r_yz`）和能量比（`e_x, e_y, e_z`）。  
   - 基于预设的四种物理模式中心（可学习）进行软分配。

3. **双聚类融合与诊断模块**  
   - 拼接时间隐特征（64维）与通道软分配权重（4维）。  
   - 经过残差块、多头自注意力、MLP 输出故障概率。  
   - 总损失 = Focal Loss + 聚类正则项。
